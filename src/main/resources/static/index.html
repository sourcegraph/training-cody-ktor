Pet Store API

Welcome to the Ktor Pet Store API. This implementation demonstrates a RESTful API using Kotlin and Ktor.

Available Endpoints:

Categories

GET /api/categories/random

Get a random category

GET /api/categories/random/{count}

Get multiple random categories

Pets

GET /api/pets/random

Get a random pet

GET /api/pets/random/{count}

Get multiple random pets

Tags

GET /api/tags/random

Get a random tag

GET /api/tags/random/{count}

Get multiple random tags

Technology Stack

  • Kotlin
  • Ktor Web Framework
  • Jackson for JSON serialization
